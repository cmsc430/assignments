# CMSC 430 PRACTICE Midterm 1, Part 4


## Instructions

You've been provided an implementation of Dupe as presented in class.

Your job is to implement `positive?`, a primitive operation that determines
whether a given integer is positive.

Update *all* relevant parts of Dupe to add the `positive?` primitive to the
language, fully supporting both the interpreter and compiler.

NOTE: You may assume inputs to `positive?` are always integers.

NOTE: there are no `positive?` tests included, so you should consider writing
some of your own.
